# tic-tac-toe

> an incredible game

## Calistirmak icin:

``` bash
# kodu cek
git clone git@github.com:Selmanh/vue-xox.git

# gerekli kutuphaneleri yukle
npm install

# hot reload ozelligiyle serveri baslat
npm run dev

# tarayicini ac ve `localhost:8080` adresine git
```
